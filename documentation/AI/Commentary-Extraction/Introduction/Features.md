# 产品功能

## 功能名称1：自动提取数据                                                
功能简介：自动挖掘评价词

功能详述： 基于机器学习算法模型和电商领域丰富的评论数据，自动挖掘评价词。                                                         
## 功能名称2：摘要生成                                                         
功能简介：可生成摘要内容

功能详述：自动分析评论关注点和评论观点，提取评论观点标签，辅助决策、生成摘要。
