# 应用场景

## 1.复工复产
各类企事业单位即将复工复产的大趋势，企事业单位多、返岗员工数量大、人员核查任务重,在复工复产过程中通过口罩佩戴识别技术，帮助企业实现对从业人员口罩佩戴情况的全过程实时监督，确保复工复产活动安全、高效运作

## 2.社区防控
社区作为防疫前线，在疫情防控进入下沉关键时期，社区人员工作任务超负荷、人工排查危险性高、社区监管存在死角等问题.通过与社区监控、关键卡点的监控、门禁设备进行口罩佩戴识别技术的集成，实现非接触排查。帮助社区完成居民逐个排查、快速摸清健康情况和流动情况，有效控制疫情蔓延

## 3.防疫检查
根据公共场所的人员管控需求，针对公共设施、出入口、检查站等人员密集场所，可使用口罩佩戴识别技术，实现秒级定位未按要求佩戴口罩的人员，联动健康码等服务。可实现人员健康信息的快速更新，全面地落实防疫措施
