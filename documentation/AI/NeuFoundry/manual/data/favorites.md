# 我的收藏（数据集）

数据集收藏功能是便于用户筛选常用的数据集，目前个人数据集及公开数据集均支持收藏。在数据集列表及数据集详情页点击"收藏"按钮，即可在"数据"—"我的收藏"中统一查看。
