# 产品功能

## 功能名称1：自然人朗读                                                                                      
功能简介：文字转声音

功能详述：可实现即输即说，高自然度，效果接近普通人的朗读水平，提供有感情、个性化的语音合成服务。
## 功能名称2：个性化定制                                                     
功能简介：用户自定义合成

功能详述：依据用户提交的音频数据，定制个性合成声音。
