# 数据集收藏

数据集收藏功能是便于用户管理常用的数据集，目前个人数据集及公开数据集均支持收藏。在数据集列表及数据集详情页点击“收藏”按钮即可，后续可以在“数据-\>我的收藏”中快速查看。
